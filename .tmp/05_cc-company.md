## 概要

Claude Code用の仮想組織管理プラグイン。`/company`コマンドで秘書が窓口となり、組織の管理と業務振り分けをサポートする。

## 主な特徴

- 秘書を中心とした階層構造で運営
- 最初は秘書室のみで開始し、業務量に応じて部署を追加する柔軟な設計
- Markdownベースのファイルシステムで組織データを管理

## ディレクトリ構成

```
cc-company/
├── .claude-plugin/           (プラグイン設定)
├── plugins/company/          (メインプラグインディレクトリ)
│   ├── skills/company/       (スキル実装)
│   └── references/           (テンプレート・参照資料)
├── docs/                     (ドキュメント)
└── README.md / LICENSE
```

## インストール方法

プラグインマーケットプレイス経由:
```
/plugin install company@cc-company
```

## 利用可能な部署

- **基本**: 秘書室
- **拡張可能**: PM、リサーチ、マーケティング、開発、経理、営業、クリエイティブ、人事

## 利用シーン

### 初期セットアップ
秘書との対話を通じて、事業内容と目標を設定するプロセスが自動化

### 日常運用
- TODOリスト管理
- メモ取得
- 業務相談
- 必要に応じて専門部署の提案（例: "リサーチ部門"）

## 技術スタック

- JavaScriptベースのプラグインアーキテクチャ
- ファイルベースの組織データ（CLAUDE.mdファイル）

## 備考

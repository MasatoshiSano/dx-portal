使い方① 社内データを調べる・分析する

Neuron・REDMINE・M365 に自然言語で聞く

📊 Neuron連携 — 社内ナレッジをAIで検索

Neuronに蓄積したKB・手順書・技術情報をLLMが理解して回答。
従来のキーワード検索では見つからなかった情報も表面化。

プロンプト例

「MELMBのEPS製部における経営課題は何か、Neuronから調べて」
「EPS制御ゲイン設定の手順をNeuronで探して要約して」

📋 REDMINE検索連携 — タスク・課題を即把握

REDMINEの全文検索と組み合わせ、プロジェクト横断で課題状況を確認。
ガントチャートも自動生成。

プロンプト例

「REDMINEで私のタスクを調べてmermaidでガントチャートも表示して」
「今週〆切が近いIssueを全プロジェクトから抽出して」

📧 M365連携 — メール・Teams・SharePointも対象

社内メール・Teams・SharePointの情報をRAGに組み込み、
「あのメールに書いてあった件は？」が即解決。

プロンプト例

「先週Teamsで相談したCANの件のまとめを教えて」
「SharePointのEPS規格書からタイムアウト仕様を抜き出して」

🖼️ マルチモーダルRAG — 画像入りドキュメントも対象

PDFや仕様書に埋め込まれた図・グラフも画像→言語化してRAG対象に。
「あの回路図の説明をして」も可能。

使い方② ハードウェアを直接動かす・評価する

AIがあなたのPCやEPS製品に直接接続して作業を代行

あなた

→

mio エージェント

🔌 できること（HSIレベル）

  • CAN / 仮想CAN を介した EPS への Vibe coding
  • UDS Diag でのDiagnostic自動操作
  • XCP 経由でのキャリブレーション操作
  • 無料のリプロツール（vFlash代替）をその場で作成・実行
  • 評価仕様書に沿った自動評価とレポート出力

📊 波形ファイル分析（Code Interpreter）

プロンプト例

「XXX.mdfにおけるI軸電流の周波数特性をFFTで評価して」
「測定データのCSVからトルクリップルを解析してグラフ出力して」

PythonをAIが書いてその場で実行。結果をグラフで即表示。

💡 ポイント： AIと EPS が直接繋がることで「仕様書を渡したらEPSが自動評価」「評価中にエラーが出たらその場で修正」が実現。 評価自動化・高度なCD活用が可能になる。

使い方③ ソフトウェア開発を加速する

コーディング支援 → レビュー → テスト生成まで一気通貫

1

コーディング支援（vibe coding / コードインタプリタ）

自然言語でコードを記述・修正。「コンパイルして、エラーがあったら修正して」と言えばAIが全部やる。
cline / roo code / continue とも連携。ローカルPC上のフォルダを直接操作。

2

コードレビュー（MELGIT-GAI 連携）

GitLab / GitHub と連携。「ローカルPCのXXフォルダの実装 diff をレビューして、添付した要求を全部満たしているか確認して」

3

テストケース自動生成

「Issue#XXでコミットしたソフトのテストケースをinstructionを基に作って」
コードEmbeddingでソースと自然言語がリンク — 関連テストを自動特定。

4

AUTOSAR対応 / 車載特化

AUTOSARの構成ファイル解析・変更提案にも対応。
SVN / GitLab / GitHub から履歴を参照しながら差分管理。

「VectorのvFlashみたいな無料でリプロできるツールを作って」

「ローカルPCのXXフォルダで実装したdiffをレビュー、

まず始めよう — Quick Start

最初の3ステップで基本機能がすぐ使える

1

Open WebUI に登録する

ブラウザから Open WebUI にアクセスしてアカウント作成。
MELGIT-GAI API キーを取得して設定すれば即利用開始。
詳細: OneNote「登録方法」ページを参照

2

まずチャットで試す

通常のChatGPTのように質問するだけでOK。
RAGを使いたい場合はナレッジベースにファイルをアップロード。
詳細: OneNote「使用方法」ページを参照

3

mio エージェントを使う

mio を選択してツールと連携したタスクを依頼する。
カスタムスキル（Agent Skills）を追加することで機能を拡張可能。
詳細: OneNote「汎用AIエージェント mio」ページを参照
